# EBRD regional groupings

Returns the names of the traditional EBRD regional groupings
(`coo_group`) that can be passed to the `group` argument of
[`list_ebrd()`](https://pablogguz.github.io/ebrdify/reference/list_ebrd.md).

## Usage

``` r
list_ebrd_groups()
```

## Value

A character vector of grouping names.

## See also

[`list_ebrd()`](https://pablogguz.github.io/ebrdify/reference/list_ebrd.md)

## Examples

``` r
list_ebrd_groups()
#> [1] "Central Asia"                       "Central Europe and Baltic States"  
#> [3] "Eastern Europe and the Caucasus"    "South-eastern Europe"              
#> [5] "Southern and Eastern Mediterranean" "Sub-Saharan Africa"                
#> [7] "Türkiye"                           
```
